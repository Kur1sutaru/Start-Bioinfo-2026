# 🧬 Python para Bioinformática — Módulo Introdutório

> Aula online de 3 horas | Google Colab | Nível: Iniciante

---

## 📋 Sobre este repositório

Material completo para a aula introdutória de Python para bioinformática, voltada a estudantes e pesquisadores que estão começando na área. Não é necessário nenhum conhecimento prévio de programação.

Todos os exercícios são executados diretamente no **Google Colab** — sem instalação!
